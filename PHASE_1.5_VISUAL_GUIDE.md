## рҹ“җ Layout Stability

### Key Principle: No Layout Shift

All animations use:
- `transform` (GPU-accelerated)
- `opacity` (GPU-accelerated)
- `backgroundColor` (smooth transition)

**Never use:**
- `width` / `height` changes
- `margin` / `padding` changes
- `display` changes

This ensures:
- No content jump
- Smooth 60fps animations
- No reflow/repaint
